# Proyecto-FILESORT

DESCRIPCIÓN
Aplicación de escritorio desarrollada en Python y wxPython para explorar, organizar y administrar archivos de una carpeta mediante una interfaz gráfica.
El objetivo del proyecto fue aplicar conceptos de Programación Orientada a Objetos y el desarrollo de interfaces gráficas utilizando wxPython.

CARACTERÍSTICAS
- Explorador de carpetas mediante árbol de directorios.
- Visualización de:
    -Nombre
    -Tipo de archivo
    -Tamaño
    -Fecha de modificación
- Apertura de archivos con doble clic.
- Buscador de archivos.
- Estadísticas del directorio seleccionado.
- Organización automática de archivos por categorías.
- Ordenamiento por fecha.
- Ordenamiento por nombre.
- Ordenamiento por tamaño (mayor a menor y menor a mayor).
- Splash Screen al iniciar la aplicación.
- Ventana "Acerca de" (About Box).

TECNOLOGÍAS UTILIZADAS
- Python 3
- wxPython
- OS
- shutil
- datetime
- wx.adv

INSTALACIÓN
Clonar el repositorio:
git clone https://github.com/carlosaricoma190-del/Proyecto-FILESORT.git

Entrar al proyecto:
cd Proyecto-FILESORT

Instalar la dependencia:
pip install wxPython

Ejecutar la aplicación:
python "FILESORT.py"
